# Arduino Water Level Controller

Automatic water level monitoring and pump control system developed using Arduino.

---

## Overview

This project implements a system that monitors the water level of a tank and automatically controls the pump.

The system operates in two modes:

Manual Mode – user can turn pump on/off  
Automatic Mode – pump activates when water level is low and stops when full.

---

## Hardware Components

Arduino Uno  
HC-SR04 Ultrasonic Sensor  
Relay Module  
Push Button  
16x2 LCD Display  
LED indicator

---

## Features

Real-time water level measurement  
Automatic pump activation  
Overflow protection  
Manual override mode  
LCD display for system status
